线性结构——线性表

补充知识点

  • Typedef: 为类型定义小名。就如你的小名是果果一样，但是指的还是你。
    例如：typedef int integer 为int类型定义了小名叫做integer。

使用int a 和使用 integer a效果是一样的

  • struct :C语言关键字，用于构建结构体
    struct name才是一个标准的结构体类型名
    注意：是类型名，而不是变量。 地位与int char类似。

  • 函数的指针和返回指针的函数

函数指针是指向函数的指针变量,在此不予解释

  • C++引用类型

使用纯C写的数据结构是不会出现引用类型的，但是大部分的教辅书都会使用引用类型来对算法进行描述

严蔚敏书中如是说

顺序表

顺序表——线性表的顺序表示与实现

  • 用一组连续的单元依次存储元素, 物理结构和逻辑结构(一致)都是顺序结构

  • 优点：
    随机存取

  • 缺点：
    空间长度需要预分配
    插入删除操作效率低
    插入：Ｅ = n / 2
    删除：Ｅ = (n-1) / 2
    平均移动表中的一半元素 n / 2

链表

  • 优点
    空间无限，具有很强扩展性
    插入删除操作时间消耗少
  • 缺点
    查找遍历很繁琐，每次都需要从头结点开始

循环链表

  • 相比于链表没有很大改变，唯一的改变在于尾指针指向了head指针所指向的区域，形成了闭环，可以从任意结点开始遍历整个链表
