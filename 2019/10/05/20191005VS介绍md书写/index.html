Markdown助力写作

发表于 2019-10-05

博客简介

  Markdown是一种极其简洁的语言，它可以很方便的书写，并且不需要担心排版——我们在书写文章的同时就是在排版。这种语言及其适合那些对格式没有太多需求，同时又强调格式，但对于word麻烦的操作很厌烦的人。

编写Markdown文档其实很简单，实际上MD（Markdown）的基本语法也就十几个，掌握起来并不难。只要多使用，每个人都可以熟练的运用。在书写MD文档之前我们只需要小小的配置一下，就可以快乐的书写了。

所需配置

  • vs code
    vs code全称“Visual Studio Code”是微软开发的一款跨平台的编译器。它支持多种语言和多种文件格式，支持代码自动补全。支持扩展，有丰富的插件可以被我们选用。对于小白来说，其实只要明白，这个东西类似于记事本，但是比记事本强大，记事本打不开的文件格式，他可以打开，记事本做不了的事，它可以做就可以了。

  • 下载VS Code
    vs code是一款开源的免费的编译器，我们并不需要购买。

    首先，我们需要在微软官网找到vs code的主页然后点击下载windows版，就可以了。

    如果你是macOS系统，则点击了解更多，进入如下页面，然后点击Download

    进入如下页面，然后选择适合的版本下载即可.

调教VS Code

  • 更换语言
    vs code设计的原生语言是英语，但是他的语言支持工作做的很好。我们只需要更改一些设置就好。
  • 语言插件安装：
    使用快捷键组合【Ctrl+Shift+p】，在搜索框中输入“configure display language”，点击它

    之后选择下图中的红色框中的插件，点击绿色的install，等待下载安装完成后，重启vs code即可。
  • 更换主题
    vs code的本来色调是黑色的，我并不喜欢，换掉它！
    我们再次打开vs code呈现下图状态

    我们选择页面右边的“自定义”中的“颜色主题”，然后就可以挑选主题颜色了，我选择的是QUite light。

    到这儿主题和语言就都设置好了。
