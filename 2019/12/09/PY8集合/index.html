PY之集合

发表于 2019-12-09 分类于 PY

集合set

集合和列表非常相似
 不同点：
    1.集合中只能存储不可变对象
    2.集合中存储的对象是无序（不是按照元素的插入顺序保存）
    3.集合中不能出现重复的元素

  • 使用 {} 来创建集合

      创建一个简单数字集合:

      s={1,2,3}
      print(s) #{1, 2, 3}
  错误示例:

      s = {10,3,5,1,2,1,2,3,1,1,1,1} #输出{1, 2, 3, 5, 10}
  启发：集合具有数学上集合的独一性，需要样本数据库中的所有数据都是不一样的时候可以使用set

      s = {[1,2,3],[4,6,7]} TypeError: unhashable type: 'list' #list为可变类型

  • 使用 set() 函数来创建集合

      s = set() # 空集合

    可以通过set()来将序列和字典转换为集合

  • 使用in和not in来检查集合中的元素

  • 集合基础方法（函数）

      len()
  使用len()来获取集合中元素的数量

      add()
  使用add() 向集合中添加元素

    updata()
    使用update() 将一个集合中的元素添加到当前集合中
    update()可以传递序列或字典作为参数，字典只能使用键



    pop()
    pop()随机删除并返回一个集合中的元素


    remove()
    remove()删除集合中的指定元素


    clear()
    clear()清空集合




    copy()
    copy()对集合进行浅复制

集合的运算

set和数学中的集合是一个数理概念，在python中的集合继承了数学中集合的几乎所有特点。

在对集合做运算时，不会影响原来的集合，对原集合不做任何改变，而是返回一个运算结果。

创建两个集合A&B

& 交集运算

result = A & B  # {3, 4, 5}

| 并集运算

result = A | B # {1,2,3,4,5,6,7}

- 差集

result = A - B # {1, 2}

^ 异或运算
获取集合（两个或多个）中不同的原素

result = A ^ B # {1, 2, 6, 7}

<=运算
检查一个集合是否是另一个集合的子集 

如果a集合中的元素全部都在b集合中出现，
那么a集合就是b集合的子集，b集合是a集合超集

<运算
检查一个集合是否是另一个集合的真子集

>=运算
检查一个集合是否是另一个的超集
>运算
检查一个集合是否是另一个的真超集
