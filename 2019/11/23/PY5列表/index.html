PY之列表

发表于 2019-11-23 分类于 PY

容器类型

容器类型指的就是列表(list)、元组(tuple)、字典(dict)、集合(set)，所谓容器就是能容纳大量个体元素的对象。
上述的所有数据类型都是Python内置的数据类型。

序列

序列是Python中最基本的一种数据结构，用于保存一组有序的数据，所有的数据在序列中都有一个唯一的位置（索引），
并且序列中的数据会按照添加的顺序来分配索引

- 序列的分类：  
    可变序列（序列中的元素可以改变）：  
        > 列表（list）  
    不可变序列（序列中的元素不能改变）：  
        > 字符串（str）      
        > 元组（tuple）

列表list

list是一种有序的集合，是Python中的一个对象，可以用来保存多个有序的数据。

    列表是用来存储对象的对象
        所谓对象，就是指内存中专门用来存储数据的一块区域，在Python中“一切皆对象”

    列表的用途
        列表主要用来存储数据，是数据的容器，可以通过列表的索引（即数据元素的下标）来操作数据对象

  • 如何创建一个列表

      通过[]就可以创建一个列表     
      my_list = [] # 创建了一个空列表
      print(my_list , type(my_list))# [] <class 'list'>

  • 列表基础内容介绍
    列表存储的数据，称为元素，一个列表中可以存储多个元素，也可以在创建列表时，指定列表中的元素

    向列表中添加多个元素时，使用,隔开 

      my_list = [10,20,30,40,50] # 创建了一个有5个元素的列表

    列表中可以保存任意的对象，包括函数、方法

      my_list = [10,'hello',True,None,[1,2,3],print]

    列表中的对象会按插入的顺序存储到列表中 

      第一个插入的对象保存到第一个位置，第二个保存到第二个位置，以此类推，同时可以通过索引（index）
  来操作列表中的元素，切记索引以零开始。索引是元素在列表中的位置，每一个元素对印一个索引,索引是
  从0开始的整数，从左到右第一个位置索引为0，第二个位置索引为1，以此类推

  • 如何获取列表元素

    通过索引获取列表中的元素

      1、语法：
          my_list[索引]    

      my_list = [10,20,30,40,50]
      my_list[0]  
      print(my_list[4]) # 50
  警告⚠
      使用的索引超过了最大的范围，会抛出异常:
      print(my_list[5]) IndexError: list index out of range

  2、列表的索引可以是负数，如果是负数则从后向前获取元素，-1表示倒数第一个，-2表示倒数第二个 以此类推

    通过len（）获取列表的长度

      len()函数，通过该函数可以获取列表的长度

  长度指的是（列表中元素的个数），获取到的长度的值也是（列表的最大索引号 + 1 ）
  my_list = [10,20,30,40,50]
  print(len(my_list)) # 5

  • 如何使用切片操作

    通过切片操作获取列表的一个子列表

      语法(1)： 
      列表名[起始:结束] 
  警告⚠
  1、通过切片获取元素时，会包括起始位置的元素，但是不会包括结束位置的元素
  2、做切片操作时，总会返回一个新的列表，不会影响原来的列表
  3、起始和结束位置的索引都可以省略不写
      如果省略结束位置，则会一直截取到最后。
      如果省略起始位置，则会从第一个元素开始截取。
      如果起始位置和结束位置全部省略，则相当于创建了一个列表的副本。  
      情况示例：
          stus = ['Ana','Tim','Bill','Juan','aploe']
          print(stus[1:]) #  ['Tim', 'Bill', 'Juan', 'aploe']
          print(stus[:3]) #  ['Ana', 'Tim', 'Bill']
          print(stus[:]) # ['Ana', 'Tim', 'Bill', 'Juan', 'aploe']
          print(stus) # ['Ana', 'Tim', 'Bill', 'Juan', 'aploe']

  语法(2):
      列表[起始:结束:步长]
          步长表示每次获取元素的间隔，默认值是1
          步长不能是0，但是可以是负数。 如果是负数，则会从列表的后部向前边取元素，如果是-1，则反置列表

      警告⚠
      print(stus[::0]) ValueError: slice step cannot be zero
      示例：
      stus = ['Ana','Tim','Bill','Juan','aploe']
      print(stus[0:5:3]) #['Ana', 'Juan']
      print(stus[::-1]) #['aploe', 'Juan', 'Bill', 'Tim', 'Ana']

  • 列表的通用操作
    1、+和*

    +可以将两个列表拼接为一个列表

    * 可以将列表重复指定的次数

    2、in和not in
    in用来检查指定元素是否存在于列表中, 如果存在，返回True，否则返回False
    not in用来检查指定元素是否不在列表中, 如果不在，返回True，否则返回False

  • 普适的列表通用函数

    len() min和max

      len()获取列表中的元素的个数

  min() 获取列表中的最小值

      max() 获取列表中的最大值

  • 普适的列表的两个方法（method）

    方法和函数基本上是一样，只不过方法必须通过 对象.方法() 的形式调用，方法实际上就是和对象关系紧密的函数

    s.index() 获取指定元素在列表中的第一次出现时索引

    s.count() 统计指定元素在列表中出现的次数

  • 修改列表中的元素

    直接通过索引来修改元素

    通过del来删除元素

    通过切片来修改列表 

      ⚠在给切片进行赋值时，只能使用序列

    通过切片来删除元素

警告⚠：
以上的所有操作，只适用于可变序列,不可变序列，无法通过索引来修改。

可以通过 list() 函数将其他的序列转换为list，然后进行修改

列表的方法

下列的方法都是适用于列表list的，只要是列表都可以调用

append()，向列表的最后添加一个元素

insert()，向列表的指定位置插入一个元素 

参数：  
1.要插入的位置  
2.要插入的元素

extend()，使用新的序列来扩展当前序列 

需要一个序列作为参数，它会将该序列中的元素添加到当前列表中  

stus.extend(['唐僧','白骨精'])  
相当于：
stus += ['唐僧','白骨精']

clear()，直接清空整个序列 

stus = ['Ana','Tim','Bill','Juan','aploe']
stus.clear()
print(stus) #[]

pop()，根据索引删除并返回被删除的元素

remove()， 删除指定值得元素，如果相同值得元素有多个，只会删除第一个

sort()，用来对列表中的元素进行排序，默认是升序排列 

如果需要降序排列，则需要传递一个`reverse=True`作为参数

⚠总结

遍历列表，指的就是将列表中的所有元素取出来，使用的是循环语句（详细见后面）

主要使用的循环语句有while和for两种，分别叙述如下

通过for循环来遍历列表，for循环的代码块会执行多次，序列中有几个元素就会执行几次。每执行一次就会将序列中的一个元素赋值给变量，所以我们可以通过变量，来获取列表中的元素。

range()是一个函数，用于生成自然数序列 

该函数需要三个参数  
    1.起始位置（可以省略，默认是0）  
    2.结束位置  
    3.步长（可以省略，默认是1）

通过range()可以创建一个执行指定次数的for循环。

for()循环除了创建方式以外，其余的都和while一样，包括else、包括break continue都可以在for循环中使用并且for循环使用也更加简单
