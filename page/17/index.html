Linux常用shell命令

发表于 2020-03-09 分类于 Linux

简介

本章介绍Linux基础操作命令便于以后查找翻阅，也供学习复习使用。

  • 环境：VM work pro15 + Ubuntu1804
  • 注意：命令输入在 -$ 后进行,一般命令输入在#后进行

    命令

  • 进入超级用户状态

      su root
  显示：root@ubuntu:/home/用户名 
  实质为切换用户命令 su

  • 查看命令ls

      ls :列出当前目录下的所有文件夹
  参数：a l    使用示例： ls -a  或 ls -l
  (1) ls -al :列出当前目录下所有文件的一个长列表，包括以句点开头“隐藏” 文件
  (2) ls a* :列出当前目录下以字母 a开头的所有文件
  (3) ls -l *.doc 给出当前目录下以 给出当前目录下以 .doc 结尾的所有文件

  • 显示当前工作路径pwd

      显示用户当前的工作路径直接输入pwd 即可.

  • 目录更改cd

      隐藏目录中:
      .  表示当前目录
      .. 表示上一级目录
  cd~       切换到主目录 
  cd /      切换到根目录 
  cd /tmp   切换到目录 /tmp
  cd Fname  切换到当前目录下的 Fname 目录
  cd ..     切换到上一级目录 
  cd ../..  切换到上二级目录 

  说明: 在使用cd/后可以使用Tab键来自动补齐文件名。

  • 清屏命令clear

      假性清屏，使用鼠标上下滑动可以找到记录

  • 显示和配置网络属性命令ifconfig

      if+config
  参数: interface 、up 、 down 、add 、netmask add

  • 新建目录mkdir和删除目录rmdir

      mkdir work 在当前目录下新建一个 work 目录

  rmdir work  将删除已存在的空目录 work

  mkdir -p this/that/work  在当前目录下建立指定的嵌套子在当前目录

  • 创建一个新文件touch

      touch demo.txt  创建一个空文件，文件名为demo.txt

  • 删除文件rm和移动文件mv

      rm 这个命令是用来删除文件的rm命令常用的参数有三个-i,-r,-f
  (1) rm –i demo      系统会询问我们是否要删除demo文件，确认后才会删除。
  (2) rm -r 目录名     删除目录，可以删除连同本目录下的子目录一起删除，功能强于rmdir
  (3)rm -f 文件名      不经确认强制删除文件

  mv 这个命令的功能是移动目录或文件引申的功能是给目录或文件重命名
  mv /tmp/xxx.tar /root     该命令将/tmp 目录下的xxx.tar 文件移动到/root 目录下
  mv aaa.tar bbb.tar        则是将当前目录下的文件 aaa.tar 更名为bbb.tar

  • cat命令

      cat 这个命令是linux 中非常重要的一个命令它的功能是显示或连结一般的ascii文本文件。
  (1) cat 文件名         显示这个文件的内容
  (2) cat file1 file2    该命令依顺序显示file1,file2 的内容
  (3) cat file1 file2>file3     该命令把file1,file2 的内容结合起来再重定向>到file3 文件中

  • 查看文件内容more

      more /etc/passwd    查看 /etc/passwd 的内容
  more命令也可以通过管道符(|) 与其他的命令一起使用

  • 搜索文件内容grep

      grep bible /etc/exports     在文件 exports 中查找包含bible的所有行
  例子:
      ① tail -100 /var/log/apache/access.log|grep 404  在 WEB服务器日志文件 access.log 的后 100 行中查找包含“ 404 ”的行
      ② tail -100 /var/log/apache/access.log|grep -v googlebot   在 WEB 服务器日志文件 access.log 的后 100 行中，查找没有被 行中，查找没有被 google 访问的行

  • 文件查找命令find

      find 命令用于查找文件系统中的指定文件其命令格式为 find 要查找的路径表达式
  例子:
      find . -name 1.txt      在当前目录及其子目录下查找文件 1.txt
      find /tmp -name 1.txt   在 /tmp 目录及其子目录下查找文件1.txt

  • 编辑文件命令vi
    vi 是Visual Interface 的简称它在Linux 上的地位就仿佛Edit 程序在DOS
    上一样它可以执行输出删除查找替换块操作等众多文本操作而且用户可以根据自己的需要对其进行定制这是其他编辑程序所没有的。vi 不是一个排版程序它不象Word 或WPS 那样可以对字体格式段落等其他属性进行编排它只是一个文本编辑程序。vi 没有菜单只有命令且命令繁多。Vi 有三种基本工作模式命令行模式文本输入模式和末行模式。

      1、命令行模式：任何时候 不管用户处于何种模式只要按一下ESC 键即可使vi 进入命令行模式
  当在shell 环境下输入vi 命令启动vi 编辑器时也是处于该模式下。

  2、文本输入模式:在命令模式下输入插入命令 i 附加命令a 打开命令 o 修改命令c 取代命令r
   或替换命令s 都可以进入文本输入模式。在该模式下用户输入的任何字符都被vi 当做文件内容
   保存起来，并将其显示在屏幕上。在文本输入过程中若想回到命令模式下按ESC 键即可。

  3、末行模式：在命令模式下用户按: 键即可进入末行模式下，此时Vi 会在显示窗口的最后一行，
  通常也是屏幕的最后一行显示一个: 作为末行模式的提示符，等待用户输入命令。多数文件管理
  命令都是在此模式下执行的（如把编辑缓冲区的内容写到文件中等）末行命令执行完后vi 自动回
  到命令模式。若在末行模式下输入命令过程中改变了主意，可按ESC 键或用退格键，将输入的命
  令全部删除之后再按一下退格键，即可使vi 回到命令模式下。

    编辑文件vi，需要先安装vim

      vi /etc/bubby.txt       用vi编辑文件/etc/bubby.txt  
  vim /etc/bubby.txt      用vi编辑文件/etc/bubby.txt  

  将 vim 挂起（暂停）：ctrl+z ctrl+z      暂停后可进行其他 ，暂停后可进行其他 shell 操作，完了之后可通fg命令切换回 vim 界面继续编辑

  :MR ： 查看历史文件记录（注意： 查看历史文件记录（注意MR 大写）
  :sp 横向切换界面窗口 
  :vsp 纵向切换界面窗口 

  快捷操作：
          切换窗口： alt+1,2
          全屏： alt+ 回车

  • 复制文件cp

      该命令类似于DOS下的copy，用法是:
  cp –r 源文件(source) 目的文件(target)
  参数 r 是指连同源文件中的子目录一同拷贝
  例子：
      cp fileName newfile.bak     把文件复制为新文件newfile.bak
      cp fileName /home/myfile    把文件复制到/home/myfile目录下
      cp * /tmp                   把当前目录下所有未隐藏文件复制到 /tmp/目录下
      cp -a docs docs.bak         递归性地把当前目录下的docs目录复制为新的目录docs.bak,保持文件属性，并复制所有的文件，包括以句点开头的隐藏文件 为了方便，-a选项包含-R选项。
      cp -i 在覆盖前 在覆盖前 询问用户
      cp -v 告诉用户正在做什么 告诉用户正在做什么

  • 压缩打包工具tar

      tar 是Linux 下最常用的打包压缩工具它在Linux 中扮演类似于WinZip 或WinRAR 在Windows 下的角色。
  下面是该工具的常用命令参数及运用举例：
  1、主要命令
  -c 创建包
  -x 解包
  -t 列出包中的内容
  -r 增加文件到指定包中
  -u 更新包中的文件
  2、可选命令
  -j 创建或解开包时 使用bzip2 进行压缩或解压
  -z 创建或解开包时 使用gzip 进行压缩或解压
  -Z 创建或解开包时 使用compress 进行压缩或解压
  -f 后面跟指定的包文件名
  -v 显示打包/解包过程
  -C 指定解包后的路径
  3、例子：
      tar -czf tmp.tar.gz /tmp        将 /tmp 目录压缩打包存放为tmp.tar.gz
      tar -xzf tmp.tar.gz -C /home    将 tmp.tar.gz 包解到/home 目录下
      tar -t tmp.tar.gz               查看 tmp.tar.gz 包中的文件信息

  • 比较文件内容diff

      diff dir1 dir2         比较目录1与目录2的文件列表是否相同，但不比较件的实际内容，不同则列出
  diff file1 file2       比较文件1与文件2的文件内容是否相同，如果文件时文本格式，则将显示不同的内容，如果是二进制代码则只表示两个文件是不同的
  comm file1 file2        比较文件，显示两个文件不相同的内容、

      reboot              重启计算机
  shutdown -r now     重启计算机，停止服务后重启
