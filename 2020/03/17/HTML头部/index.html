HTML头部

发表于 2020-03-17 分类于 HTML

<head>

什么是HTML的头部呢？在整个HTML文档中由<head></head>标签包裹的所有内容都称为HTML的头部。

  • HTML头部内包含的所有内容都不会在浏览器页面中显示，<title>元素包含的内容会在地址栏中显示。
  • <head>头部中常常会包含一些元数据（Metadata）——描述数据的数据。

    元数据<meat>

    HTML使用<meta>元素方式为一个文档添加元数据。不同种类的 <meta>元素可以被包含进页面的<head>元素中。

<meta>元素包含了name 和 content 属性：

  • name 指定了meta 元素的类型； 说明该元素包含了什么类型的信息。
  • content 指定了实际的元数据内容。

引用文件<link> <script>

  • 网站图示（favicons）: 给网页添加自定义图标。

  • <link> 元素经常位于文档的头部,用于引用CSS（文档样式表）

  • <script>元素常位于文档的<body>身体部分，用于引用Javascript文件

将JS文件放于`</body>’之前，而不是文档的头部的好处是，确保在加载脚本之前浏览器已经解析了HTML内容。

URL

Write Time Data： 2020-03-18

URL（uniform resource locator）统一资源定位器，是一个定义了在网络上的位置的一个文本字符串。例如Mozilla的英文主页定位在https://www.mozilla.org/en-US/。URL可以指向HTML文件、文本文件、图像、文本文档、视频和音频文件以及可以在网络上保存的任何其他内容。

绝对URL和相对URL？

  • 绝对URL： 指向由其在Web上的绝对位置定义的位置，包括 protocol(协议) and domain name（域名）
  • 相对URL： 指向与您链接的文件相关的位置，更像文件的路径。
  • 绝对并不比相对好，相对也不一定比绝对佳，合适的地方使用合适的方法。

    超链接

    超链接就是一个指向其他网页或文档甚至文档指定部分，经由点击（或激活）就可以跳转到指定页面或文档部分的连接器。

    链接的解析

  • 链接标签<a>，包含属性href title
  • 将文本转化为链接（普通链接），href属性将包含您希望链接指向的网址,使用title属性添加支持信息。

I'm creating a link to to my blog .

  • 块级链接，你可以将一些内容转换为链接，甚至是块级元素。比如图片；

  • 如何返回上级目录：返回上一个目录级 使用两个英文点号表示 ..

      如果需要的话，你可以将这些功能的多个例子和复杂的url结合起来。
  例如：../../../complex/path/to/my/file.html

  • 链接到文档片段：首先必须给要链接的元素（通常为HTML元素）分配一个id属性，然后才可以链接，使用#链接

  • download属性，提供一个默认的保存文件名。

超链接的作用

  • 创建一个导航栏菜单
    通常在主页上的导航栏中会包含其他的附属网页，此时可以使用超链接（相对链接）来导航到其他网页。
  • 电子邮件链接
    最常用的形式mailto:link （链接/邮箱地址），链接简单说明收件人的电子邮件地址。
