Python2424_字典集合

发表于 2020-01-25 分类于 Python

字典（dict）

 字典属于一种新的数据结构，称为映射（mapping）
 字典的作用和列表类似，都是用来存储对象的容器
 列表存储数据的性能很好，但是查询数据的性能的很差

在字典中每一个元素都有一个唯一的名字，通过这个唯一的名字可以快速的查找到指定的元素。 

 在查询元素时，字典的效率是非常快的。

在字典中可以保存多个对象， 这个对象，我们称其为值（value）。每个对象都会有一个唯一的名字。这个唯一的名字，称为键（key），通过key可以快速的查询value. 

    key:value  
    所以字典，我们也称为叫做键值对（key-value）结构
    每个字典中都可以有多个键值对，而每一个键值对我们称其为一项（item）

创建字典

可以使用 {} 来创建字典

    d = {} # 创建了一个空字典

创建一个保存有数据的字典 

    语法：  
        {key:value,key:value,key:value} 
    说明：     
        字典的值可以是任意对象  
        字典的键可以是任意的不可变对象（int、str、bool、tuple ...），一般都会使用str
        字典的键是不能重复的，如果出现重复，后边的会替换到前边的

根据键来获取值 

print(d['name'],d['age'],d['gender'])

如果使用了字典中不存在的键，会报错KeyError 

print(d['hello']) KeyError: 'hello'

使用 dict()函数来创建字典

每一个参数都是一个键值对，参数名就是键，参数名就是值
（这种方式创建的字典，key都是字符串）

该函数也可以将一个包含有双值子序列的序列转换为字典 

双值序列，序列中只有两个值，[1,2] ('a',3) 'ab'
子序列，如果序列中的元素也是序列，那么我们就称这个元素为子序列，[(1,2),(3,5)]

len() 获取字典中键值对的个数

d = dict(name='孙悟空',age=18,gender='男') 
print(len(d)) #3

in not in函数

in 检查字典中是否包含指定的键
not in 检查字典中是否不包含指定的键 

print('hello' in d)

获取字典中的值

原理：根据键来获取值

通过[]来获取

语法：
    d[key]

栗子:

 d = dict(name='孙悟空',age=18,gender='男') 
 print(d['name'])

通过[]来获取值时，如果键不存在，会抛出异常 KeyError

get(key[, default])

该方法可根据键来获取字典中的值 

- 如果获取的键在字典中不存在，会返回None 
- 也可以指定一个默认值，来作为第二个参数，这样获取不到值时将会返回默认值
    print(d.get('name'))
    print(d.get('hello','默认值'))

修改/添加

  • d[key] = value 如果key存在则覆盖，不存在则添加

setdefault(key[, default])

可以用来向字典中添加key-value 

如果key已经存在于字典中，则返回key的值，不会对字典做任何操作
如果key不存在，返回key值，并向字典中添加这个key，并设置value

update([other])

将其他的字典中的key-value添加到当前字典中
如果有重复的key，则后边的会替换到当前的（更新）

删除

可以使用 del 来删除字典中的 key-value

popitem()

随机删除字典中的一个键值对，一般都会删除最后一个键值对 

- 删除之后，它会将删除的key-value作为返回值返回
- 返回的是一个元组，元组中有两个元素，第一个元素是删除的
key，第二个是删除的value
- 当使用popitem()删除一个空字典时，会抛出异常 
KeyError: 'popitem(): dictionary is empty'

pop(key[, default])

根据key删除字典中的key-value 会将被删除的value返回！

- 如果删除不存在的key，会抛出异常
- 如果指定了默认值，再删除不存在的key时，不会报错，而是直接返回默认值
result = d.pop('d')
result = d.pop('z','这是默认值')

clear()用来清空字典

复制

copy() 该方法用于对字典进行浅复制 

- 复制以后的对象，和原对象是独立，修改一个不会影响另一个
- 注意，浅复制会简单复制对象内部的值，如果值也是一个可变
对象，这个可变对象不会被复制

遍历

keys() 该方法会返回字典的所有的key
该方法会返回一个序列，序列中保存有字典所有的键

  • 通过遍历keys()来获取所有的键

values() 该方法会返回字典的所有的value
该方法会返回一个序列，序列中保存有字典所有的值

  • 通过遍历values()来获取所有的值

items() 该方法会返回字典中所有的项
它会返回一个序列，序列中包含有双值子序列
双值分别是，字典中的key和value

集合（set）

 集合和列表非常相似
 不同点：
    1.集合中只能存储不可变对象
    2.集合中存储的对象是无序（不是按照元素的插入顺序保存）
    3.集合中不能出现重复的元素

创建集合

使用 {} 来创建集合

创建一个简单数字集合:

s={1,2,3}
print(s) #{1, 2, 3}

错误示例:

s = {10,3,5,1,2,1,2,3,1,1,1,1} #输出{1, 2, 3, 5, 10}

启发：集合具有数学上集合的独一性，需要样本数据库中的所有数据都是不一样的时候可以使用set

s = {[1,2,3],[4,6,7]} TypeError: unhashable type: 'list' #list为可变类型

使用 set() 函数来创建集合

s = set() # 空集合

可以通过set()来将序列和字典转换为集合

检查元素

使用in和not in来检查集合中的元素

集合基础方法（函数）

len()

使用len()来获取集合中元素的数量

s = {'hello'}
print(len(s)) #4

add()

使用add() 向集合中添加元素
s = {‘hello’}
s.add(10)
s.add(30)
print(s) #{10, 30, ‘hello’}

updata()

使用update() 将一个集合中的元素添加到当前集合中
update()可以传递序列或字典作为参数，字典只能使用键

pop()

pop()随机删除并返回一个集合中的元素

& 交集运算

result = A & B  # {3, 4, 5}

| 并集运算

result = A | B # {1,2,3,4,5,6,7}

- 差集

result = A - B # {1, 2}

^ 异或运算

获取集合（两个或多个）中不同的原素

result = A ^ B # {1, 2, 6, 7}

<=运算

检查一个集合是否是另一个集合的子集 

如果a集合中的元素全部都在b集合中出现，
那么a集合就是b集合的子集，b集合是a集合超集

<运算

检查一个集合是否是另一个集合的真子集

>=运算

检查一个集合是否是另一个的超集

>运算

检查一个集合是否是另一个的真超集
