PY面向对象

发表于 2020-01-25 分类于 PY

对象?面向对象?类?

什么是对象？什么是面向对象？
什么是类？
不要着急，本章我们就来说一下什么是对象，什么是类，又如何面向对象编程。

面向对象编程

Python是一门面向对象的语言,同时也提供了函数式编程的方法，可以继续使用面向过程式编程的方法，这是毫无问题的。但是却失去了python本来的意义，既然面向对象，那我们就该使用面向对象的思想————简单理解就是语言中的所有操作都是通过对象来进行的，通过抽象、分类的思维方式将生活中的实际问题转化为数学对象，通过操作对象来完成任务目标。面向对象其实质是一种思维方式，不再以面向过程的编程方法将关注点放在任务完成的步骤上，而是放在为完成任务所需要的对象（参与者）上，通过对对象的操作来完成既定目标。

请牢记：
        面向对象编程，重要的不是步骤，而是对象。

面向对象编程的优点是：易阅读，易维护，易复用；缺点是：过于抽象，难于编写。

  • 什么是对象：（三种定义）

      1、对象是内存中专门用来存储数据的一块区域。
  2、对象是人们要进行研究的任何事物，它不仅能表示具体的事物，还能表示抽象的规则、计划或事件。
     对象具有状态，一个对象用数据值来描述它的状态。对象还有操作，用于改变对象的状态，对象及其
     操作就是对象的行为。对象实现了数据和操作的结合，使数据和操作封装于对象的统一体中。

          状态：操作  ==  属性：方法

  3、通过类定义的数据结构实例。

  • 什么是类：

      1、类是对象的图纸；类是一个用来创建对象的对象。类是type类型的对象，定义类实际上就是定义了一个type类型的对象。
  2、类是用来描述具有相同的属性和方法的对象的集合，它定义了该集合中每个对象所共有的属性和方法，一个对象就是一个类的实例。

  • 什么是实例：类在实例化之后叫做一个实例

       “类”是静态的，不占进程内存，而“实例”拥有动态内存。而在数据库中数，则代表一些程序的集合。

  实例和对象基本上是同义词，它们常常可以互换使用。
      对象代表了类的一个特定的实例。对象具有身份（identity）、类型（type）、和属性值（attribute values）3个特征。实例是对象的具体表示，操作可以作用于实例，实例可以有状态地存储操作结果。实例被用来模拟现实世界中存在的、具体的或原型的东西。
      对象就是类的实例，所有的对象都是实例，但并不是所有的实例都是对象。例如，一个关联（UML关系中的一种）的实例不是一个对象，它只是一个实例、一个连接。我们常见的实例都是类的实例，此时二者没有区别。除了类的实例外的实例都不是对象。

  • 什么是属性|什么是方法：属性是类中定义的变量，方法是类中定义的函数

定义一个类

类和对象都是对现实生活中的事物或程序中的内容的抽象

实际上所有的事物都由两部分构成：
    1.数据（属性）
    2.行为（方法）

在类的代码块中，我们可以定义变量和函数

变量会成为该类实例的公共属性，所有该类实例都可以通过    对象.属性名    的形式访问
函数会成为该类实例的公共方法，所有该类实例都可以通过    对象.方法名()  的形式调用方法

注意：

方法调用时，第一个参数由解析器自动传递，所以定义方法时，至少要定义一个形参！这点会在后面着重讲解。

问：实例为什么能访问到类中的属性和方法

类中定义的属性和方法都是公共的，任何该类实例都可以访问

    属性和方法查找的流程：
        当我们调用一个对象的属性时，解析器会先在当前对象中寻找是否含有该属性，
            如果有，则直接返回当前的对象的属性值；
            如果没有，则去当前对象的类对象中去寻找，如果有则返回类对象的属性值；
            如果类对象中依然没有，则报错！

类对象和实例对象中都可以保存属性（方法）

    如果这个属性（方法）是所有的实例共享的，则应该将其保存到类对象中；
    如果这个属性（方法）是某个实例独有，则应该保存到实例对象中。    

    一般情况下，属性保存到实例对象中，而方法需要保存到类对象中。

Python中使用class关键字来定义类

语法：
    class 类名([父类]) :

        公共的属性...

        # 对象的初始化方法
        def __init__(self,...):
            ...

        # 其他的方法    
        def method_1(self,...):
            ...

        def method_2(self,...):
            ...

        ...

方法中self参数详解：
类中的方法每次被调用时，解析器都会默认传递一个实参（不管你定义了参数与否），且该参数总是第一个被传递。该参数其实就是调用方法的对象本身，一般我们都会将该参数命名为self。

使用类创建对象

我们说过对象是类的实例，只定义类而不是用的话是毫无意义的，使用类的方法就是利用类来创建对象————类的实例化。

  • 类的实例化：通常把用类创建对象的过程称为实例化。

      一般而言，类的实例化和直接使用类的格式是不一样的。

类的实例化相当于模板之余产品的概念，类是一个模板，类的实例化就相当于将模板制作称为一个个产品（对象）。

Python中有内置对象，此前的字符串、数字、数组、字典都是内置对象，都具有其相应的类。譬如说 int() float() bool() str() list() dict()。当我们调用数组的length方法的时候，实际上就是在调用该类型的内置函数。

创建对象的一般流程：

1.创建一个变量
2.在内存中创建一个新对象
3.将对象的id赋值给变量

对于该流程因该要很熟悉，且要明白，在Python中通常第一二三步是同步完成的，如上面的mc = Myclass()，如果是在java语言中第二步的体现就是new一个新方法，mc = new Myclass()。

提问？一个类可以创建多个对象吗？类与对象之间有什么关系呢？

先答：类可以创建多个对象.

类与对象的关系： 一个类可以创建多个对象，多个对象可以共享一个类。类中的属性和方法对于使用该类创建的所有对象都是通用的。类中的属性方法改变，所有的实例对象中的属性方法会发生对应的改变，而一个实例对象中的类和方法的改变并不会影响其他的对象，也不会对类造成改变。

提问？ 如何在一个类的方法中使用类的属性？

类中属性和方法的修改

类中的方法称为类方法，实例对象中的方法称为实例方法（对象方法），实例方法中无法修改。

初始化函数__init__(self)

初始化函数又称为构造函数、或者是魔术方法。这是在类中定义的一些特殊方法，特殊方法都是以开头，以结尾的方法，特殊方法不需要我们自己调用，不要尝试去调用特殊方法，特殊方法将会在特殊的时刻自动调用。

学习特殊方法我们需要了解的就是：
                                1.特殊方法什么时候调用
                                2.特殊方法有什么作用
1、什么时候调用
    特殊方法在我们将对象的id赋值给变量之前调用
    整套流程：
        1.创建一个变量
        2.在内存中创建一个新对象
        3.__init__(self)方法执行
        4.将对象的id赋值给变量

2、有什么作用
    每一个魔术方法都有他独特的作用，都是设计来优化编程体验的。

如果我们需要了解更多，请查看Python Docs

作用：init会在对象创建以后立刻开始执行，可以用来向新创建的对象中初始化属性，调用类创建对象时，类后边的所有参数都会依次传递到init()中。
